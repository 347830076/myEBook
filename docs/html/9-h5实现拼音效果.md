# h5 实现拼音效果

```html
<!-- ruby:定义 ruby 注释（中文注音或字符）。 -->
<!-- rt: 定义字符（中文注音或字符）的解释或发音。 -->
<!-- rp: 在 ruby 注释中使用，以定义不支持 ruby 元素的浏览器所显示的内容。 -->
   <ruby>
    广<rt>guang</rt>
    东<rt>dong</rt>
  </ruby>

  <ruby>
    漢 <rp>(</rp><rt>han</rt><rp>)</rp>
    字 <rp>(</rp><rt>ji</rt><rp>)</rp>
  </ruby>
```
