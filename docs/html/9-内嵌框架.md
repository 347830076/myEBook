# 内嵌框架 iframe

HTML 内联框架元素 表示嵌套的浏览上下文，有效地将另一个 HTML 页面嵌入到当前页面中。

属性说明： 

- width 宽度 
- height 高度
- frameborder 边框 取值为1时（默认值），告诉浏览器在当前iframe与其他iframe之间绘制边框，取0时则无需绘制此边框。
