# 折叠列表

`<details>` 标签用于描述文档或文档某个部分的细节。

open属性定义details折叠信息一开始是否可见。

`<summary>` 标签包含 details 元素的标题。

`<details>`和`<summary>`目前只支持Chrome和Safari，兼容性低。 

```html
<details open="open">
  <summary>关于你中奖的秘密</summary>
  <p>想太多了</p>
  <p>哈哈哈</p>
</details>
```
