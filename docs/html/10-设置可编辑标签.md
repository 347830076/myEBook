# 设置可编辑标签

**contenteditable 属性设置标签是否可编辑**

_注意：如果元素未设置 contenteditable 属性，那么元素会从其父元素继承该属性。_

```html
<p contenteditable="true">我是p段落，可是我也能编辑</p>
```
