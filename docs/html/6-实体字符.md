# HTML 实体字符

**在 HTML 中，某些字符是预留的。例如：在 HTML 中使用小于号(<)和大于号(>)，浏览器会误认为它们是标签。如果想要正确显示预留字符，就可以使用实体字符。**

一些在键盘上找不到的字符也可以使用字符实体来替换。

字符实体有三部分：一个与号`&`和一个实体名称（或者一个`#` 和一个实体编号），以及一个分号`;`。

**实体名称对大小写敏感！**
