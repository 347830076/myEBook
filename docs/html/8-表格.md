## 表格 table 元素

表格由` <table> `标签来定义。每个表格均有若干行（由` <tr>` 标签定义），每行被分割为若干单元格（由` <td>` 标签定义）

一个简单的例子如下：

```html
<table border="1">
    <tr>
        <td>第一行，单元格1</td>
        <td>第一行，单元格2</td>
    </tr>
    <tr>
        <td>第二行，单元格1</td>
        <td>第二行，单元格2</td>
    </tr>
</table>
```

## 属性

- width
- height
- align
  - left 左边 （默认）
  - center (居中)
  - right (右边)
- border  边框宽度
- cellspacing 单元格的间距
- cellpadding 表格边框与内容之间的间距

## 包含标签

  - table 表格
  - caption 表标题
  - tbody 表格内容标签
  - tr 行
  - th 表头单元格， 通常会呈现为居中的粗体文本
  - td 单元格
